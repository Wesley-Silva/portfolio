Imagens

Ecommerce Enterprise

Aplicação web de comércio eletrônico com autenticação de usuários (login/registro por e-mail), catálogo de produtos, carrinho de compras, e checkout com múltiplas opções de pagamento (cartão, PayPal etc). Funcionalidades incluem escolha de método de entrega, confirmação de pedido via e-mail e acompanhamento do status da compra. Tecnologias utilizadas: ASP.NET Core MVC, Entity Framework Core, SQL Server, integração com gateways de pagamento, envio de e-mails automatizado.

Gerador de Senha

Sistema para geração de senhas aleatórias com regras configuráveis pelo usuário, como tamanho, tipos de caracteres (letras, números, símbolos) e complexidade. Desenvolvido em ASP.NET Core com C#, focado em segurança e usabilidade.

Máquina de refrigerante(Simulador)

Simulador de máquina de refrigerante desenvolvido com HTML, CSS e JavaScript, que permite ao usuário selecionar bebidas, inserir valores e receber troco virtualmente. Projeto focado em lógica de programação, manipulação de DOM e experiência interativa no navegador.
